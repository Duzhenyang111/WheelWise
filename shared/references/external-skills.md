# External Skills and Reference Resources

WheelWise may reference or optionally depend on external skill ecosystems, but it must not copy their content into this repository.

## UI UX Pro Max

UI UX Pro Max is the design intelligence reference for ui-demo and visual-brief work.

- GitHub: https://github.com/nextlevelbuilder/ui-ux-pro-max-skill
- Skills listing: https://skills.sh/nexu-io/open-design/ui-ux-pro-max
- Suggested use: design intelligence reference for `ui-demo` and `visual-brief`, including UI direction, information architecture, design-system thinking, style selection, component/layout reasoning, and anti-pattern review.
- WheelWise role: pass product type, user, surface, modules, constraints, product strategy, demo needs, and HTML display requirements to a UI/UX skill when visual, demo, or `wheelwise-report.html` planning is needed.
- Boundary: do not copy UI UX Pro Max content into WheelWise. Capture only WheelWise-specific adaptation notes and final decisions.
- HTML display: UI UX Pro Max may guide layout, hierarchy, color, typography, and anti-pattern checks for `wheelwise-report.html`, but the HTML content must still come from WheelWise's Chinese Markdown report.
