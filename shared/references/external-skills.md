# External Skills and Reference Resources

WheelWise may reference or optionally depend on external skill ecosystems, but it must not copy their content into this repository.

## UI UX Pro Max

- GitHub: https://github.com/nextlevelbuilder/ui-ux-pro-max-skill
- Skills listing: https://skills.sh/nexu-io/open-design/ui-ux-pro-max
- Suggested use: UI direction, information architecture, design-system guidance, and UI anti-pattern review.
- WheelWise role: pass product type, user, surface, modules, and constraints to a UI/UX skill when UI planning is needed.

## product-on-purpose/pm-skills

- GitHub: https://github.com/product-on-purpose/pm-skills
- Suggested use: PRD, user stories, acceptance criteria, ADR, competitive analysis, persona, market sizing, and experiment design references.
- WheelWise role: reference product-management structure while preserving WheelWise's build/buy/reuse decision layer.

## github/awesome-copilot

- GitHub: https://github.com/github/awesome-copilot
- Suggested use: architecture, security, compliance, supply-chain, and codebase-understanding references.
- WheelWise role: reference checklist dimensions while keeping WheelWise's own `risk-review` and reuse evaluation decisions.
