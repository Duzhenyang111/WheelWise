# WheelWise

WheelWise is a Superpowers-style multi-skill pack for pre-development product and technical decisions.

Call the main skill:

```text
Use $using-wheelwise to evaluate this idea: ...
```

The main skill routes the work through internal skills for idea intake, delivery surface strategy, feasibility review, build/buy/reuse/fork/reference analysis, risk review, optional parallel research, and Codex-ready execution planning.

Version 1 focuses on the core loop:

1. Structure the raw idea.
2. Choose the delivery surface.
3. Judge feasibility.
4. Decide Build / Buy / Reuse / Fork / Reference by module.
5. Review key risks.
6. Produce a New Product Brief and Codex-ready execution plan.

External skills such as UI UX Pro Max, pm-skills, and awesome-copilot are reference resources or optional dependencies only. WheelWise does not copy their content.
